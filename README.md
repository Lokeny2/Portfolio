# Moses Ekutan | Portfolio Website

## Overview

A responsive portfolio website built with Bootstrap 5 to showcase my projects, skills, and journey as a frontend developer.

## Live Demo

https://lokeny2.github.io/Portfolio/

## Technologies Used

- HTML5
- CSS3 (with custom styles and theming)
- Bootstrap 5.3
- Bootstrap Icons
- Google Fonts (Inter + Playfair Display)

## Sections

- **Navbar** – Fixed, responsive navigation with smooth scrolling
- **Hero** – Full-screen section with introduction, CTA buttons, and social links
- **Projects** – Card grid showcasing my work (includes Tanit Coding School landing page)
- **About** – Bio, profile image, and skills badges
- **Contact** – Form with fields for name, email, subject, and message
- **Footer** – Social links and copyright

## What I Learned

- Building responsive layouts with grid system in Bootstrap
- Customizing Bootstrap with CSS variables and overrides
- Creating a consistent color theme using CSS custom properties
- Implementing smooth scrolling and fixed navigation
- Form handling with proper `name` attributes and accessibility
- Using Bootstrap Icons for social links and visual elements
- Structuring a multi-section single-page portfolio

## Installation

1. Clone the repository:
   git clone https://github.com/Lokeny2/Portfolio
2. Open `index.html` in your browser
